Overview

Processing is an open source programming language and IDE built for the electronic arts, new media art, and visual design communities with the purpose of teaching the fundamentals of computer programming in a visual context.

This summer, I took Introduction to Computational Arts and Creative Programming for Digital Media & Mobile Apps on COURSERA and learnt processing. I also bought Learning Processing and Processing语言权威指南 to help me have a deeper understanding of it. These 3 demos are based on the sample codes which were given in Creative Programming for Digital Media & Mobile Apps.
